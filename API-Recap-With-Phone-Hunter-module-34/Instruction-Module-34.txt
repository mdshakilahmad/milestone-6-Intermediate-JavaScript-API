যত কাজ আছে আজকে সব ফোনে ফোনে করে ফেলার দিন (মডিউল ৩৪)

#Module_release #Intermediate_js_milestone #Module_34



আজকে আরেকটু ভালো করে API প্রাকটিস করবে। শুধু ডাটা লোড না। তার সাথে সাথে মজার কিছু জিনিস করবে। আগের চাইতে API আরেকটু ভালোভাবে ক্লিয়ার হবে। আরেকটু বুঝতে পারবে। এরপরেও টেনশন করার কিছু নাই। টেনশন না করে আজকের জিনিসগুলো একাধিকবার প্রাকটিস করে ফেলবে। দেখবে কিভাবে কি করেছি। এবং ধীরে ধীরে তুমি Fetch এবং API রিলেটেড কাজগুলো করতে করতে কিছু এরর খেতে খেতে জিনিসগুলাতে অভ্যস্ত হয়ে উঠবে। তার আগে একটু বেখাপ্পা লাগতেই পারে। সেটা কোন ব্যাপার ই না। এছাড়াও জায়গামতো আমরা হালকা করে Async Await ও মেরে দিয়েছি। কি আছে জীবনে। 



.



আজকের মডিউল 

ফোন রিলেটেড একটা ওয়েবসাইট হবে আজকে। অর্থাৎ একটা API দেয়া আছে। সেটা থেকে Data লোড করবে API ইউজ করে। তারপর সেই ডাটা অনুসারে Phone এর ছবি এবং বিভিন্ন ডাটা দেখাবে। শুধু সেটাই না। একটা সার্চ বক্স বসিয়ে ফেলবে তোমার সাইট এর মধ্যে। এরপর সেই ফিল্ডের মধ্যে সার্চ ফাংশনালিটি ইমপ্লিমেন্ট করে ফেলবে। আবার যদি এমন কিছু সার্চ দিয়ে ফেলো যেটা নাই, তাহলে কোন ফোন নাই সেই টাইপের মেসেজ দেখাবে। আবার যদি কিছু ফোন পাওয়া যায়, তাহলে সেটা দেখাবে। 



মাঝে মধ্যে ডাটা লোড হতে যদি সময় লাগে। তাহলে একটা চমৎকার Loading Spinner চলে আসবে। আবার যখন ডাটা লোড হয়ে দেখিয়ে ফেলবে তখন সেই লোডিং স্পিনার গায়েবুল হাওয়া হয়ে যাবে। আবার সব ডাটা দেখাতে না চাইলে, জাস্ট স্লাইস করে অল্প কয়েকটা ডাটা দেখাবে। এরপর আবার Show All নামক বাটনে চাপ দিলে সব ডাটা দেখিয়ে দিবে। কি মজা না?



তবে মজা এখানেই শেষ না। বরং সব ফোন এর মধ্যে Show Details নামক বাটন যোগ করবে। সেই বাটনে ক্লিক করলে Dynamic ভাবে সেই ফোন এর জন্য API থেকে Data Load করবে। সেই ডাটা কনসোল এ লগ করে দেখবে। তবে শুধু কনসোল এ দেখলেই হবে না। বরং Modal এ ওপেন করে দেখাবে। তাহলে সেই লেভেলের মজা হবে। কি রেডি তো? তাহলে আর দেরী কেন? চলো শুরু করে দিই আজকের মডিউল।
