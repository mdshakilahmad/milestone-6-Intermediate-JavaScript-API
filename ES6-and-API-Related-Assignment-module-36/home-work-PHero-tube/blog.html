1. Discuss the scope of var, let, and const Chat gpt

var আমরা use করবো না let এবং const use করবো । এটাই same নাম দিইয়ে বার বার variable তৈরি করা জায়।

var হচ্ছে redeclared and reassigned হয় এটাই প্রব্লেম হয়।

আর let দিইয়ে same scope এ same নামের variable বানানো যাবে না। let দিইয়ে reassigned করতে পারব but cannot be redeclared within the same scope.

constants: Also introduced in ES6, const is used to declare constants, whose values cannot be re-assigned.

এটা ব্লক scope const এমন জায়গায় use করবো যেখানে কখনো তার নামের পরিবর্তন হবে না।

32-2 module এ বিস্তারিত আছে।

2. কোন কিছু নেই যেটা আমরা ডিক্লিয়ার করিনি সেটা হচ্ছে undefined । undefined মানে কোনো কিছু নেই
আর null হচ্ছে যদি বুঝাত চাই যে এখানে কিছুই নেই তাহলে null দিবো । অর্থাৎ যেখানে undefined আসবে সেখানে যদি বুঝাতে চাই যে এখানে কিছুই নেই তাহলে null সেট করে দিবো ।
undefined দেখানো যাবে না ।

3. REST API হচ্ছে server থেকে ডাটা এনে দেখানো আমাদের ব্রাউজারে । এবং আমরা চাইলে আমাদের ডাটা পাঠাতেও পারি।
